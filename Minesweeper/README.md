## Minesweeper
AI Plays Minesweeper

![Demo](demo.gif)

### How does it work?
AI updates its knowledge base as the game progresses to infer and make safe moves if possible.

### How to use it?
`$ git clone https://github.com/Praful932/AI-Projects.git`

`$ cd AI-Projects/Minesweeper`

To run the game

`$ python runner.py`
